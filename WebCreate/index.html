О нас

Web Create - бесплатный конструктор сайтов, который позволяющий быстро создать веб-страничку из нескольких простых составляющих. Кулинарный сайт, резюме, интервью, статьи - это лишь малый перечень из того, что вы можете сделать с помощью этого инструмента.

Работая, вы можете не беспокоиться за положение элементов на сайте - редактор сам соберёт их в столбец. Также присутствует возможность глубокого редактирования дизайна разных объектов: кнопок, изображений, строк ввода и так далее.

Web Create, в отличие от Wix и Word Press, не так функционален, однако прост и быстр. Также, данный редактор частично учит пользователя работе с сайтом.

Проект находится в состоянии бета, однако, мы амбициозно настроены и имеем множество идей для улучшения нашего сайта:

  • Написать краткий обзор в язык CSS, который позволит неопытному пользователю работать с дизайном на сайте.
  • Сделать адаптивность редактора и сайтов, которые на нём создаются.
  • Добавить больше настроек и элементов, при этом, не усложнить работу.
  • Сделать главное меню сайта, где пользователь сможет создавать, удалять и производить другие махинации с проектами.
  • Добавить функцию сбора информации на готовом сайте, чтобы было возможным составлять различные формы для отправки: анкеты, голосования и т.д.
  • Улучшить дизайн редактора.

С уважением, команда DoneWeb.
